## Meme-Fusion AI

Meme-Fusion AI is a hybrid meme studio that lets users combine two or more recognizable meme formats and generate a new fusion with AI. The product is positioned around three connected layers:

- AI meme fusion
- SocialFi ranking and remix mechanics
- Web3 launch paths through Four.meme on BNB Chain

The current build is a polished Next.js prototype with wallet connection, branded landing experience, and product surfaces for fusion, SocialFi, vault, and launch views.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Wagmi
- RainbowKit
- Viem

## Local Development

Run the app locally:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Product Summary

Users choose meme references, the system analyzes structure and cultural cues, and AI produces a hybrid meme concept designed to feel both new and instantly legible. The SocialFi layer adds voting, remixing, and reputation mechanics, while top-performing hybrids can move into a launch-ready token concept via Four.meme.
